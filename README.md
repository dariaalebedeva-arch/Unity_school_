# Unity School Landing

![Status](https://img.shields.io/badge/status-active-success)
![Stack](https://img.shields.io/badge/stack-HTML%20%7C%20CSS%20%7C%20JS-blue)

Лендинг языковой школы «Юнити» (Новосибирск) с акцентом на современный дизайн, анимации и конверсионные блоки.

## О проекте

Проект представляет собой одностраничный сайт для школы английского и немецкого языков.  
Лендинг собран на чистом `HTML + CSS + JavaScript` без фреймворков и подходит для быстрого запуска, хостинга на статических платформах и дальнейшей кастомизации.

## Цели лендинга

- Презентовать школу и методику обучения
- Показать преимущества и направления подготовки (ОГЭ/ЕГЭ, разговорный, базовый)
- Снять основные возражения потенциальных учеников
- Собирать заявки через форму обратной связи

## Технологии

- `HTML5` — структура страницы
- `CSS3` — адаптивная верстка, визуальный стиль, анимации
- `Vanilla JavaScript` — интерактив и валидация формы
- `Google Fonts` — типографика (`Cormorant Garamond`, `IBM Plex Sans`, `Roboto Flex`)

## Реализованные блоки

- Hero-экран с CTA и фото преподавателя
- Блоки целевой аудитории, программ и целей обучения
- Пошаговый процесс обучения
- Информация о школе и преподавателях
- Методика и результаты обучения
- Блок с сомнениями и FAQ
- Отзывы (карточки)
- Форма заявки с валидацией
- Финальный конверсионный CTA-блок

## Анимации и интерактив

- Scroll reveal для секций и карточек
- Stagger-анимация для списков
- Анимированные CTA-кнопки
- Параллакс-движение декоративных элементов в hero
- Эффект `Variable Proximity` для главного заголовка (изменение `font-variation-settings` по курсору)
- Поддержка `prefers-reduced-motion`

## Структура проекта

```text
Unity_School/
├─ index.html        # Основная разметка лендинга
├─ styles.css        # Все стили и анимации
├─ animations.js     # Интерактив, reveal, валидация формы
├─ promt.md          # Исходное ТЗ по контенту
├─ SKILL.md          # Внутренние дизайн-ориентиры

```

## Локальный запуск

1. Скачайте или клонируйте репозиторий.
2. Откройте `index.html` в браузере.

Для корректной разработки рекомендуется запуск через локальный сервер (например, расширение Live Server в VS Code/Cursor).

## Форма заявки

Форма в текущей версии работает на фронтенде:
- проверяет обязательные поля,
- форматирует телефон,
- показывает статусы ошибок/успеха.

Чтобы получать реальные заявки, подключите backend или внешний webhook:
- Telegram Bot API
- Google Sheets / Apps Script
- EmailJS / Formspree
- CRM webhook
